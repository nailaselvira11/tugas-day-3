Axios adalah salah satu library JavaScript populer yang banyak digunakan untuk membuat HTTP requests di aplikasi web. Axios perlu diinstall, "npm install axios", cara mengirim GET Requests harus menggunakan axios.get(). metode ini mengambil URL sebagai parameternya dan mengembalikan PROMISE yang diselesaikan dengan data respons. untuk mengirim post request menggunakan axios.post(), untuk mengirim PUT Requests diperlukan axios.put(). Selanjutnya untuk mengirim DELETE requests. Selain metode yang diebutkan tadi, kita dapat mengirim HTTP requests untuk me retrieve, create, update dan delete data from APIs
axios punya beberapa kelebihan dari fetch Saat mengambil data dari tempat yang salah akan menampilkan pesann error 404.
